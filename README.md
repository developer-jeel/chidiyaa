# Chidiyaa

### A Modern Social Media Platform Inspired by Instagram

Chidiyaa is a full-stack social media web application inspired by Instagram, built using Django. Users can create accounts, upload posts, like and comment on content, follow other users, chat in real-time, and explore a beautiful modern UI with dark/light themes and smooth animations.

---

## Features

### Authentication System

* User Registration
* Login / Logout
* Forgot Password
* Secure Authentication
* Profile Management

### Social Media Features

* Upload Posts with Images
* Like & Unlike Posts
* Comment System
* Save Posts
* Share Posts
* Stories Feature
* Follow / Unfollow Users
* Explore Feed

### Chat System

* Real-time Messaging UI
* Chat Rooms
* Send & Receive Messages
* User-to-User Conversations

### Notifications

* Like Notifications
* Follow Notifications
* Comment Notifications

### Modern UI/UX

* Instagram Inspired Design
* Responsive Layout
* Dark & Light Theme Toggle
* Glassmorphism Effects
* Smooth Animations
* Interactive Components

### Advanced Features

* Search Users
* Suggested Profiles
* Reels Style Section
* Infinite Scroll
* AJAX-Based Interactions
* Modern Dashboard

# Installation

## 1. Clone Repository

```bash id="6jzgze"
git clone https://github.com/yourusername/chidiyaa.git
```

---

## 2. Go to Project Folder

```bash id="g8wz4h"
cd chidiyaa
```

---

## 3. Create Virtual Environment

```bash id="lx7hdg"
python -m venv env
```

---

## 4. Activate Virtual Environment

### Windows

```bash id="55b8r7"
env\Scripts\activate
```

---

## 5. Install Dependencies

```bash id="f49e0i"
pip install -r requirements.txt
```

---

## 6. Run Migrations

```bash id="3tyn61"
python manage.py migrate
```

---

## 7. Start Server

```bash id="dml88x"
python manage.py runserver
```
